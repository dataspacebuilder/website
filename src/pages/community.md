# Join the Builders

Dataspace Builders is a global community of developers, architects, cloud providers, and organizations advancing trusted data sharing. Here's how to get involved.

---

## Get Connected

### Eclipse Dataspace Working Group

The [Eclipse Dataspace Working Group](https://dataspace.eclipse.org/) is the home of the Dataspace Protocol standards and Eclipse Dataspace Components.

- Industry collaboration on standards
- Technical working groups
- Governance and roadmap discussions

### GitHub

All components are open source and developed on GitHub:

- [**Eclipse EDC**](https://github.com/eclipse-edc) — Core components
- [**Connector**](https://github.com/eclipse-edc/Connector) — Control and Data Plane
- [**Identity Hub**](https://github.com/eclipse-edc/IdentityHub) — Decentralized identity
- [**Federated Catalog**](https://github.com/eclipse-edc/FederatedCatalog) — Catalog aggregation

## How to Contribute

We welcome contributions in many forms:

### Code Contributions

- Bug fixes and feature improvements
- New extensions and data planes
- Performance optimizations
- Test coverage improvements

### Documentation

- Tutorials and guides
- Architecture explanations
- Translation to other languages
- Example use cases

### Community Support

- Answer questions in GitHub Discussions
- Share your implementation experiences
- Present at community calls
- Write blog posts
